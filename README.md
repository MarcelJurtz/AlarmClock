# Android AlarmClock

[![Icon On Hold](https://raw.githubusercontent.com/MarcelJurtz/ProjectStatusIcons/master/OnHold.png)](https://github.com/MarcelJurtz/ProjectStatusIcons)

This app is a android-alarmclock which is able to set alarms based on wished duration or a specific time. The clock is also going to alarm the user, if the battery is below a defined level to prevent oversleeping by forgetting to plug in the charger.
